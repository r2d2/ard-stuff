ard-dht

BOM :

  • Arduino
  • DHT11 or DHT22 sensor
  • 433Mhz emitter module

Hardware :

  • Arduino Digital 3 : sensor data pin
  • Arduino Digital 4 : RF emitter data pin
  • Arduino Digital 13 : led pin (optional)
